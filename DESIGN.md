# 在庫管理Webアプリ 設計書

## 概要
冷蔵庫や日用品の在庫を管理し、買い物が必要なアイテムを一目で確認できるWebアプリ。

## 主な機能

### 1. 在庫管理
- アイテムの追加・編集・削除
- カテゴリ分け（冷蔵庫、日用品、その他）
- 現在の在庫数を管理
- 最小在庫数（閾値）を設定

### 2. 買い物リスト表示
- 在庫数が最小在庫数を下回ったアイテムを自動でハイライト
- 「買うもの」タブで購入が必要なアイテムだけを一覧表示

### 3. データ永続化
- LocalStorageを使用してブラウザにデータを保存

## 技術スタック
- HTML5
- CSS3（レスポンシブ対応）
- Vanilla JavaScript（フレームワーク不使用でシンプルに）
- LocalStorage（データ永続化）

## データ構造

```javascript
{
  items: [
    {
      id: "uuid",
      name: "牛乳",
      category: "refrigerator", // refrigerator | daily | other
      quantity: 1,
      minQuantity: 2,
      unit: "本",
      createdAt: "2026-01-31T00:00:00Z",
      updatedAt: "2026-01-31T00:00:00Z"
    }
  ]
}
```

## UI設計

### レイアウト
```
┌─────────────────────────────────────┐
│  🏠 在庫管理                         │
├─────────────────────────────────────┤
│  [すべて] [買うもの🛒] [+追加]       │
├─────────────────────────────────────┤
│  🥬 冷蔵庫                          │
│  ┌─────────────────────────────────┐│
│  │ 牛乳    🔴 1/2本  [-][+][✏️][🗑]││
│  │ 卵      🟢 6/6個  [-][+][✏️][🗑]││
│  └─────────────────────────────────┘│
│  🧴 日用品                          │
│  ┌─────────────────────────────────┐│
│  │ トイレットペーパー 🔴 1/3個     ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

### 色分け
- 🔴 赤: 在庫不足（quantity < minQuantity）
- 🟢 緑: 在庫十分（quantity >= minQuantity）
